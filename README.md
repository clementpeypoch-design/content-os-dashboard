# ContentOS Dashboard

Content management dashboard for social media.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Netlify

```bash
npm install -g netlify-cli
netlify login
netlify init
netlify deploy --prod
```

Or connect your GitHub repo directly in the Netlify dashboard — the `netlify.toml` handles all build configuration automatically.

## Sections

- **Instagram** — Kanban board for managing posts
- **Analytics** — Performance metrics with charts
- **Calendar** — Monthly content calendar
- **Competitors** — Track competitor accounts
- **News** — RSS feed aggregator
